# Fitit iOS

This directory is generated by Capacitor.

## Production API

The bundled WebView uses `VITE_API_BASE_URL` from the Vite build.

For production iOS builds, create `frontend/.env.production` with:

```env
VITE_API_BASE_URL=https://your-backend.example.com/api
```

Then run:

```bash
npm run ios:sync
```

Without `VITE_API_BASE_URL`, web builds use the browser-relative `/api` path. That is useful when Flask serves the web build, but a native iOS WebView needs an absolute backend URL for login and data APIs.
